# training-notebooks
<h4>This repository contains Databricks notebooks that provide training resources and specific examples. To setup this notebook within your Databricks environment:</h4>

<ol>
<li>Download the repository's content (i.e. notebook and the associated images)</li>
<li>Import the notebook into your Databricks workspace</li>
<li>Using the Databricks CLI, copy the images into a folder within DBFS.</li>  
  - Currently, the filepath for the images in the notebook is in dbfs:/FileStore/nancy/learningPathsNB/  which must be referred to as /files/nancy/learningPathsNB/ in the notebook<br>
  - Change the filepaths in the notebook to where your images are stored also
<li> Adjust the permissions of the notebook so that the intended users have read only permissions to it</li>
